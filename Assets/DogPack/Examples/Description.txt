Dog model contains 42 seperate animations and 7 blendshape emotions.
You can modify any color of the texture easily by changing the colors in the psd file provided.
All animations are also included in 3ds studio max file.
If you want to mod animation or add your own and in doubt check this manual section 
"Importing Animations using multiple model files"
https://docs.unity3d.com/Manual/Splittinganimations.html
